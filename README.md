# 📂 File Organizer Automation Script

## 🚀 Overview

This project is a Python-based automation tool that organizes files into folders based on their file extensions.

For example:

- .txt → TXT_Files
- .jpg → JPG_Files
- .pdf → PDF_Files
- .mp4 → MP4_Files

The script automatically creates folders and moves files into their respective categories.

---

## ✨ Features

- Organizes files automatically
- Creates folders based on file type
- Supports user input
- Generates operation logs
- Uses exception handling
- Easy to use

---

## 🛠 Technologies Used

- Python

- File Handling
- Exception Handling

## ▶️ How to Run

1. Open terminal

```bash
python organizer.py
```

2. Enter folder path

```text
test_folder
```

3. Files will be organized automatically.
